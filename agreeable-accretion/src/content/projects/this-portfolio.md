This site is intentionally styled like an early-2000s editorial web publication instead of a generic modern portfolio. The goal was to make it feel personal and opinionated rather than interchangeable.

## Focus

- static generation with Astro
- minimal client-side JavaScript
- custom CSS with a retro editorial layout

## Outcome

The portfolio is fast, easy to host, and now supports markdown-based writing for both blog posts and project entries.
